# Référence Finisseur — Vocabulaire et champs métier

## Terminologie correcte

| À utiliser | À éviter |
|---|---|
| dorure à chaud / marquage à chaud | "hot stamping" (sauf si interlocuteur anglophone) |
| feuille dorée / feuille argentée / feuille pigmentée | "papier dorure" |
| estampage / embossage (relief sans encre) | "gaufrage" (terme moins précis) |
| débossage (creux sans encre) | "anti-gaufrage" |
| vernis sélectif UV (brillant / mat / soft-touch) | "vernis partiel" |
| sérigraphie (directe / en relief) | "impression spéciale" |
| découpe laser / découpe emporte-pièce | "découpe à forme" |
| pelliculage métallisé / holographique | "pellicule spéciale" |
| contre-collage | "assemblage de supports" |
| zone de marquage (L × H mm) | "endroit à dorer" |

## Informations délai

- Date limite de réception des supports imprimés chez le finisseur

## Notes spécifiques

- Pour la **dorure à chaud** : indiquer si le support est pelliculé (mat de préférence pour l'adhérence)
- Pour le **vernis sélectif** : préciser si la zone de vernis correspond exactement à un tracé vectoriel ou à une zone libre
- Pour les **clichés** : si réutilisation d'un cliché existant, fournir la référence et vérifier sa compatibilité avec le support actuel
- Pour la **sérigraphie** : indiquer le nombre de passages (= nombre de couleurs) et préciser si impression en relief (puff) souhaitée
- Pour la **découpe laser** : fournir le fichier vectoriel DXF ou AI — le finisseur ne travaille pas à partir d'images raster
