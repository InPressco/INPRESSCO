# Référence Papetier — Vocabulaire et champs métier

## Terminologie correcte

| À utiliser | À éviter |
|---|---|
| feuilles en format (L × H mm) | "feuilles A4/A3" (sauf si c'est le format exact) |
| grammage (g/m²) | "épaisseur" |
| couché mat / couché brillant | "papier brillant / mat" |
| offset non couché | "papier blanc simple" |
| bouffant / main | "papier épais" |
| opacité (%) | "papier qu'on voit pas en transparence" |
| sens fibre (sens machine / sens travers) | "sens du papier" |
| rame de 500 feuilles / bobine de Xkg | "paquet de feuilles" |
| papier recyclé certifié FSC / PEFC | "papier éco" |
| teinte (blanc naturel / blanc bleuté / crème / ivoire) | "couleur du papier" |

---

## Informations complémentaires utiles

- Mentionner si le papier est destiné à une impression **offset** ou **numérique** (Indigo / toner) → impact sur la compatibilité du support
- Pour les supports spéciaux (métallisé, pearlescent) : indiquer si une impression est prévue et avec quelle technologie
- Préciser si une **fiche technique** (datasheet) du papier est nécessaire pour validation client

---

## Notes fournisseur

- Toujours demander le **délai de disponibilité** (stock ou commande spéciale)
- Demander si un **échantillon** peut être envoyé avant commande pour les supports spéciaux
- Pour les grosses quantités : demander les conditions de **livraison franco** vs départ usine
