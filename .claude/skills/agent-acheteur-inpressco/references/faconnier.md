# Référence Façonnier — Vocabulaire et champs métier

## Terminologie correcte

| À utiliser | À éviter |
|---|---|
| feuilles imprimées / feuilles vierges | "papier brut" |
| format de coupe / format fini | "taille" |
| dos carré collé (DCC) | "reliure parfaite" |
| dos carré cousu (DCS) | "reliure couture" |
| pelliculage mat / brillant / soft-touch | "plastification" |
| rainage / rainurage | "pliage préparatoire" |
| découpe à façon | "découpe sur mesure" |
| cahiers / signatures | "groupes de pages" |
| faux-titre, garde, tranche | termes normalisés — utiliser tels quels |
| marque de coupe (bleed) | "débord" |

---

## Bloc technique façonnage

```
Type de produit : [livre / brochure / carnet / boîte / chemise / étui / autre]
Format fini (L × H mm) :
Format de coupe (L × H mm) :
Nombre de pages :
Type de reliure : [dos carré collé / dos carré cousu / agrafé centre / spirale / reliure japonaise / autre]
Couverture :
  - Grammage couverture : [g/m²]
  - Pelliculage couverture : [mat / brillant / soft-touch / aucun]
  - Marquage spécial couverture : [dorure / embossage / découpe laser / aucun]
Intérieur :
  - Grammage intérieur : [g/m²]
  - Impression intérieur : [N&B / quadri / bicolore]
Façonnage complémentaire :
  - Rainage : [oui / non]
  - Perforation : [oui / non — si oui, préciser]
  - Encartage / assemblage : [oui / non]
  - Conditionnement : [vrac / sous cellophane / en carton]
Fichiers prêts à imprimer : [oui / non / à confirmer]
```

---

## Informations délai

Préciser :
- Date souhaitée de livraison des feuilles imprimées au façonnier (BAT inclus)
- Date souhaitée de livraison du produit fini chez le client final
- Si vernis/pelliculage fourni par le façonnier ou livré avec les feuilles

---

## Notes spécifiques

- Mentionner si les feuilles sont **déjà imprimées** ou si l'impression est incluse dans la consultation
- Si pelliculage ou marquage à chaud : préciser si In'Pressco gère cette étape ou si c'est sous-traité par le façonnier
- Pour les produits avec cahiers : indiquer le nombre de cahiers et le nombre de pages par cahier
